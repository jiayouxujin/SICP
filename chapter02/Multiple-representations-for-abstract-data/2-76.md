# generic operations with explicit dispatch

不管是新增加一个类型或者操作，都需要修改系统的整体代码

# data-directed

主要特点：

1. 上层不需要知道自己调用的具体函数，会通过**数据类型**和**操作**在op-tables里查询

不管是新增加一个类型还是通用操作，都不需要修改上层的代码。只需要在op-table里操作就可以

# message-passing

主要特点：

1. 与data-directed不同，这里的"op-table"被纵向分割了
2. 操作跟对象绑定在一起【面向对象的思想】

新增加一个类型---不需要修改系统代码

新增加一个操作---需要修改该对象的操作，并且所有已经实例化的对象都需要重新实例化，才能够使用该操作
